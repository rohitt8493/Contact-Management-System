# Contact Management System

A simple Flask contact management app for CRUD operations, search, and basic styling. Single-file or multi-file setup supported; stores contacts in `contacts.json` for simple persistence.

---

## Features
- **Create Read Update Delete** contacts  
- **Search** by name or phone number  
- **File persistence** using `contacts.json`  
- **Minimal UI** with CSS served from `static` or a route  
- **Easy to run** with Python and Flask

---

## Requirements
- **Python 3.8 or newer**  
- **Flask**
